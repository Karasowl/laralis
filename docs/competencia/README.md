## Cómo Agregar un Nuevo Competidor

1. Crear carpeta `docs/competencia/{nombre-competidor}/`
2. Agregar `ANALISIS.md` con:
   - Historia y fundadores
   - Métricas (clientes, países, empleados)
   - Modelo de negocio
   - Fortalezas y debilidades
3. Agregar `FEATURES.md` con lista detallada de funcionalidades
4. Actualizar este README
